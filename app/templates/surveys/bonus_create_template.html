Step 2 — Survey template

Google Form template

Use the following Google Form as the base for your Bonus Survey. This link will always create a copy that you can customize.

Template (DO NOT EDIT TOKEN FIELD)
https://docs.google.com/forms/d/1zRtyHIuKXB7xk_Cz4O_ZkF14L5JPLzXfrxLe1_u7BaU/copy

What to do

  1. Edit the questions to fit your research goals.
  2. Use Pre-fill form to generate a survey URL. This step identifies where the token belongs in the link.
    In the top left 3 Dot menu > Pre-Fill Form
    Use Get pre-filled link — then copy the generated URL.
  3. In the new window, at the very bottom of the page locate the field labeled “User Token (DO NOT EDIT)”. Enter the placeholder value: user_token_here
    The token field must exist, be required, and contain user_token_here.
  4. After you have entered user_token_here in the token field, click Get link. Then click COPY LINK
    Get Link will generate a link for you
    Click COPY LINK to copy the link containing entry.<id>=user_token_here.
  5. This is what a valid link looks like
    https://docs.google.com/forms/d/e/1FAIpQLSeI9AZ0WGIs5eEIuwlz9rMbax6ck79cKWDIRZW0e7WwkUQtMA/viewform?usp=pp_url&entry.1711341715=user_token_here

    Your link must include user_token_here. The system will replace this value for each participant.

  6. Invalid link. Please check the instructions above.
✔ Correct: Token field exists and is required.
✖ Do not: Delete or rename the token field.
