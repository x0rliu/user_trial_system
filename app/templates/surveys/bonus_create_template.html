What to do

  1. Edit the questions to fit your research goals.
  2. Use Pre-fill form to generate a survey URL. This identifies where the token belongs in the link.
    In the top-left three-dot menu, choose Pre-fill form.
    Use Get pre-filled link, then copy the generated URL.
  3. In the new window, scroll to the field labeled User Token (DO NOT EDIT). Enter this placeholder value: user_token_here
    The token field must exist, be required, and contain user_token_here.
  4. After entering user_token_here, click Get link, then click Copy link.
    Get Link generates the pre-filled survey URL.
    Copy the link containing entry.<id>=user_token_here.
  5. This is what a valid link looks like:
    https://docs.google.com/forms/d/e/1FAIpQLSeI9AZ0WGIs5eEIuwlz9rMbax6ck79cKWDIRZW0e7WwkUQtMA/viewform?usp=pp_url&entry.1711341715=user_token_here

    Your link must include user_token_here. The system replaces that value for each participant.
